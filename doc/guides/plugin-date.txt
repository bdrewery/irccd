== Introduction

The plugin `date` shows the current date with a specified format. The date
is set from the server.

== Installation

The plugin `date` is distributed with `irccd`. To enable it add the following to your
`plugins` parameter:

----
plugins = "date"
----

== Usage

The plugin `date` reacts to special command, you just need to write `!date` and
if you want a specific format `!date format` where *format* is same as the C
strftime(3) function.

.Example
----
markand: !date
irccd: markand, 10/26/13 10:56
markand: !date %H:%M
irccd: markand, 10:57
----

The default format is `%m/%d/%y %H:%M`.
