Get the parent path of a file. This function may be used to get the parent path of a file. Example: calling dirname on "/foo/bar/test.txt" will returns "/foo/bar".

Get the time spent from start. Get how many milliseconds spent since the irccd startup.

mkdir (path)
Create a directory. Create recursively a directory, like mkdir -p. This function will return two values on error.

Parameters:

  • path the path

Returns:

  1. true on success or false plus an error
  2. the error message
opendir (path, skipSpecial)
Open a directory. This function will open the directory specified by path. Optional boolean skipSpecial may be passed to avoid ".", ".." for being treated.

Parameters:

  • path the path
  • skipSpecial forget "." and "..", default: false

Returns:

    a Directory object

see also:

    Directory
splitUser (user)
Split the nickname and host. Extract the nickname and host from a user, for instance with foo!~foo@localhost, foo will be returned as first value, localhost as second.

Parameters:

  • user the user to split

Returns:

  1. the nickname
  2. the hostname
usleep (msec)
Sleep for milliseconds. Suspend the execution thread.

Parameters:

  • msec the number of milliseconds

Class Date

Date
Date:calendar ()
Get a calendar. Return a table with the following fields:
  • seconds (int): seconds elapsed [0-60]
  • minutes (int): minutes elapsed [0-59]
  • hours (int): hours elapsed [0-23]
  • month (int): number of month [1-12]
  • year (int): the real year

Returns:

    a table
Date:format (format)
Format a date. Format the date with a specific format, see strftime(3) for keywords.

Parameters:

  • format the format

Returns:

    the date formatted
Date:timestamp ()
