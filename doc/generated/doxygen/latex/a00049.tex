\subsection{Detailed Description}
Backward compatibility. 

This class is used to wrap the parser and its parameters because the class \hyperlink{a00048}{Parser} now throw an exception in the constructor but the Lua A\-P\-I expect that .new return and object and .open returns the error. So we only create the parser at .open method.
