Server:getIdentity ()
Get the identity used for this server. Returns a table with the following fields:
  • name (string): the identity name
  • nickname (string): the nickname currently in use
  • username (string): the username
  • realname (string): the full name
  • ctcpversion (string): the CTCP Version reply

Returns:

    an identity table
Server:getName ()
Get the server name id.

Returns:

    the name id
Server:getInfo ()
Get the server information. Returns a table with the server connection information with the following fields:
  • name (string): the server ident
  • hostname (string): the hostname
  • port (int): the port
  • ssl (bool): true if using SSL
  • sslVerify (bool): true if using SSL verification

Returns:

    the information table

Server:names (channel, func)
Get the list of names. This is a deferred call, the function func must have the following signature and will be called when irccd get the results.

function callback(list)

where list is a sequence table with all nicknames.

Parameters:

  • channel the channel to check
  • func the function

See also:

    Deferred.lua

Usage:

     server:names(
	"#test",
	function (list)
		-- Play with list sequence
	end
 )

Server:whois (target, func)
Get whois information from a user. This is a deferred call, the function func must have the following signature

function callback(info)

where info is a table with the following fields:

  • nickname (string): the nickname
  • user (string): the username
  • host (string): user's hostname
  • realname (string): the realname
  • channels (table): a sequence table of all channels

Parameters:

  • target the nickname target
  • func the function

See also:

    Deferred.lua

Usage:

     server:whois(
	"test",
	function (info)
		-- Play with info
	end
 )
