# socket.new

*Available since [[version-1-1|1.1]]*

Create a socket.

## Function

### SYNOPSIS

````lua
s, err = function socket.new(family, type)
````

### ARGUMENTS

- family, one of the [[irccd-socket-Family|socket.family]]
- type, one of the [[irccd-socket-Type|socket.type]]

### RETURNS

- s, the socket or nil on failure (see [[irccd-Socket|Socket]])
- err, the error message
