Send some data. This function sends data to a stream based socket.

# Synopsis

````lua
nsent, err = function Socket:send(data [, address])
````

# Arguments

- data, the data
- address, the address if UDP

# Returns

- nsent, the number of bytes sent or nil
- err, the error message
