Get all the data. The iterator returns one value which may be any type, except function and userdata. It removes all values from the pipe.

# Synopsis

````lua
iterator = function Pipe:list()
````

# Returns

- iterator, an iterator function

# Example

````lua
-- Get all data
for v in p:list() do
    print(tostring(v))
end
````
