---
method: Directory.prototype.remove
---

Synonym of [Directory.remove(recursive)][1] but the path is taken from the
directory object.

# Synopsis

````javascript
Directory.prototype.remove(recursive)
````

# Arguments

- recursive, recursively or not (default: false)

# Throws

- Any exception on error
